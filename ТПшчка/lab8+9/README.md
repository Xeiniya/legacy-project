# Разработка приложений на языках Swift и Objective-C: геолокация, MapKit и CoreData

## Описание
Данный репозиторий содержит примеры и задания по разработке iOS-приложений с использованием технологий геолокации, MapKit и CoreData на языках Swift и Objective-C. Проекты демонстрируют работу с хранением данных, авторизацией пользователей, определением местоположения, картами и базами данных.

## Структура репозитория
Репозиторий содержит следующие ветки:

1. **main** — основная ветка с описанием проекта и ссылкой на отчет
2. **example-task1** — пример приложения с хранением данных в PLIST на Swift
3. **example-task2** — пример приложения с авторизацией и хранением данных в NSUserDefaults на Swift
4. **example-task3** — пример приложения с функцией геолокации на Swift
5. **example-task4** — пример приложения с определением местоположения по карте на Swift
6. **example-task5** — пример системы бронирования билетов на Objective-C с CoreData и MapKit
7. **example-task6** — пример приложения с использованием базы данных Core Data
8. **example-task7** — пример приложения прогноза погоды
9. **feature-task2-1** — приложение "Библиотека" с хранением данных в NSUserDefaults и PLIST
10. **feature-task2-2** — приложение для бронирования билетов на маршрутный автобус с MapKit, CoreLocation и CoreData
11. **testing-task2-1** — тестирование приложения "Библиотека" с хранением данных в NSUserDefaults и PLIST

## Примеры

### Пример 1.1: Хранение данных в PLIST на Swift
Демонстрация работы с Property List (PLIST) для хранения и чтения структурированных данных в iOS-приложениях.

### Пример 1.2: Авторизация с NSUserDefaults на Swift
Реализация формы авторизации/регистрации с сохранением пользовательских данных с помощью NSUserDefaults.

### Пример 1.3: Геолокация на Swift
Использование фреймворка CoreLocation для определения текущего местоположения пользователя.

### Пример 1.4: Определение местоположения по карте на Swift
Интеграция MapKit и CoreLocation для отображения местоположения пользователя на карте.

### Пример 1.5: Система бронирования билетов на Objective-C
Приложение для бронирования авиабилетов с использованием CoreData для хранения данных и MapKit для выбора маршрута.

### Пример 1.6: Работа с Core Data
Пример приложения для управления списком студентов с возможностью добавления, удаления и сохранения данных в Core Data.

### Пример 1.7: Приложение прогноза погоды
Интеграция с API OpenWeatherMap для получения данных о погоде с поддержкой геолокации и ручного ввода города.

## Задания

### Задание 2.1: Приложение "Библиотека"
Разработано iOS-приложение на Swift с:
- Интерфейсом авторизации (хранение в NSUserDefaults)
- Выводом данных из PLIST в UICollectionView
- Подробным просмотром описания книг
- Локализацией на 3 языка (русский, английский + третий на выбор)
- Архитектурой MVC

### Задание 2.2: Система бронирования автобусных билетов
Разработано iOS-приложение на Swift с:
- Выбором маршрута на карте (MapKit)
- Просмотром списка рейсов (UITableView)
- Интеграцией с API OpenWeatherMap для прогноза погоды
- Хранением данных в CoreData
- Локализацией на 3 языка
- Поддержкой архитектуры MVVM

## Установка
1. Клонируйте репозиторий: `git clone https://github.com/fpmi-tp2025/tpmp-gr9b-lab8-Xeiniya.git`
2. Перейдите в нужную ветку: `git checkout example-task1` (или другую интересующую вас ветку)
3. Откройте соответствующий .xcodeproj файл в Xcode
4. Для проектов с зависимостями выполните `pod install` (если используется CocoaPods)

## Требования
- Xcode 13 или новее
- iOS 15+
- Для некоторых проектов требуется API ключ OpenWeatherMap
